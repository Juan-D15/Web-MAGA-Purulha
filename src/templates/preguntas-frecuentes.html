Preguntas Frecuentes

Encuentra respuestas a las preguntas más comunes sobre el sistema

¿Qué es el Sistema de Gestión de Eventos MAGA Purulhá?

Es una plataforma web diseñada para gestionar y administrar eventos, proyectos y actividades de la Oficina Regional MAGA Purulhá. Permite el registro, seguimiento y reporte de todas las actividades realizadas en las diferentes comunidades y regiones.

¿Quién puede usar el sistema?

El sistema está disponible para todo el personal de la Oficina Regional MAGA Purulhá. Los usuarios pueden tener diferentes niveles de acceso según su rol (Administrador, Personal, etc.). Algunas funciones están disponibles para usuarios no autenticados.

¿Cómo puedo crear una cuenta?

Las cuentas de usuario son creadas por los administradores del sistema. Si necesitas una cuenta, contacta con el administrador del sistema o con tu supervisor.

Uso del Sistema

¿Cómo agrego un nuevo evento o proyecto?

Para agregar un nuevo evento, necesitas tener permisos de administrador. Accede al menú "Gestiones" y selecciona "Agregar Evento". Completa el formulario con la información requerida y guarda los cambios.

¿Cómo registro un avance de un evento?

Para registrar un avance, accede a la página del evento específico y utiliza la opción "Agregar Avance". Puedes incluir descripción, fecha, evidencia fotográfica y otros detalles relevantes.

¿Puedo editar o eliminar un evento después de crearlo?

Sí, los eventos pueden ser editados y eliminados por usuarios con permisos de administrador. Accede al menú "Gestiones" y selecciona "Editar Evento" para realizar modificaciones.

Reportes

¿Cómo genero un reporte?

Para generar reportes, necesitas tener permisos de administrador. Accede al menú "Gestiones" y selecciona "Crear Reporte". Elige el tipo de reporte que necesitas y configura los filtros según tus requerimientos.

¿Puedo exportar los reportes?

Sí, los reportes pueden ser exportados en diferentes formatos según el tipo de reporte. Busca la opción "Exportar" en la página de reportes para descargar el archivo.

Problemas Técnicos

¿Qué hago si olvidé mi contraseña?

Si olvidaste tu contraseña, contacta con el administrador del sistema para que pueda restablecerla. Por seguridad, las contraseñas no pueden ser recuperadas directamente desde el sistema.

¿Qué navegadores son compatibles?

El sistema es compatible con los navegadores modernos más populares, incluyendo Google Chrome, Mozilla Firefox, Microsoft Edge y Safari. Se recomienda usar la última versión disponible de estos navegadores.

¿Cómo reporto un problema o error en el sistema?

Puedes reportar problemas o errores utilizando el formulario de "Asistencia Técnica" en la página de Configuración General. Describe el problema en detalle para que podamos ayudarte de la mejor manera.

¿No encontraste tu respuesta?

Si tienes alguna pregunta adicional o necesitas asistencia, no dudes en contactarnos. Puedes usar el formulario de asistencia técnica o contactar directamente al equipo.
